## Featured Projects

### Nokia NBUC: Generative AI Security Pipeline

An AI-powered security analysis pipeline designed to identify vulnerabilities in containerized applications and Kubernetes deployments.

Key features:

- Container image vulnerability scanning using Trivy
- Kubernetes manifest analysis using Kyverno, Kubescape, and kube-score
- AI-powered remediation suggestions using an LLM
- RAG-based security chatbot for querying scan results
- CLI and Flask-based web interface

---

### DishaRakshak: GPS-less Navigation Device for Soldiers

A GPS-independent navigation system concept designed for soldiers operating in environments where GPS may be unavailable, unreliable, or jammed.

Key features:

- Foot-mounted IMU units for step and motion tracking
- Belt-mounted central processing unit
- Dead reckoning-based path estimation
- ESP32-based communication between modules
- Simulation-based path visualization

---

### Self-Care Web App

- Goal selection for fitness, skincare, and focus
